How to use this app

To use this app, follow these steps:

  1. Type or paste the message you want to encrypt or decrypt in the input box.
  2. Select the cryptographic algorithm you want to use from the drop-down menu.
  3. If applicable, choose the key you want to use for the encryption or decryption.
  4. Click the "Encrypt" or "Decrypt" button, depending on what you want to do.
  5. The result will be displayed in the output box.
